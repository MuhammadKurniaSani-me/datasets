# Beijing Multi-Site Air Quality Data

> This dataset includes hourly air pollutants data from 12 nationally-controlled air-quality monitoring sites.

## Overview

The air-quality data originates from the **Beijing Municipal Environmental Monitoring Center**. Meteorological data for each air-quality site is matched with the nearest weather station from the **China Meteorological Administration**.
